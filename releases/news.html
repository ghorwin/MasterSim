Aktuelles

Release 0.9.3 veröffentlicht (13.01.2022)

Es gab etliche kleinere Verbesserungen, die im Wesentlichen mit dem Debuggen von FMUs zu tun hatten. Außerdem ist MasterSim jetzt in einem Debian-Package Repository für Ubuntu und abgeleitete Distros erhältlich. Insgesamt wurde vor allem die Linux Integration vervollständigt - sowohl für die alleinstehende Variante als auch das Debian-Paket.

Release 0.9.0 veröffentlicht (08.09.2021)

Zahlreiche kleinere Verbesserungen der Benutzbarkeit, vor allem für Linux und Mac-Anwender (siehe Changelog).

Patch-Release 0.8.3 veröffentlicht (24.11.2020)

Dieses Release bringt viele kleine Verbesserungen und nun auch ein komplettes deutsches Handbuch. Das Handbuch kann aus dem Programm heraus im Hilfe-Menü aufgerufen werden, oder wie bisher von der MasterSim Dokumentationsseite geladen werden.

Patch-Release 0.8.2 veröffentlicht (14.11.2019)

Es ist vollbracht - ein 60 seitiges Nutzerhandbuch (als PDF) oder als Online-Ausgabe steht zur Verfügung. Darin sind alle Details und 'Geheimnisse' der MasterSim-Programmoberfläche und des Simulators beschrieben. Das Handbuch ist auf der MasterSim Dokumentationsseite verfügbar. Außerdem sind dort nun die bisher zu MasterSim entstandenen Publikationen veröffentlicht.

Im neuen Release sind auch viele kleinere Fehler behoben worden (alles, was beim Schreiben des Handbuches aufgefallen ist) und eine schöne kleine Erweiterung, die sicher gefällt (hoffe ich jedenfalls).

"Kleines" Patch-Release 0.8.1 veröffentlicht

Nur 2 Tage nach dem 0.8er Release gibt es in der neuen Version einige kleine, aber vielleicht für den einen oder anderen wichtige Erweiterungen. Es können Transformationsoperationen zwischen Ausgangs- und Eingangsvariablen (Vorzeicheninvertierung, Einheitenumrechnung) angegeben werden und die Oberfläche ist nun deutlich schneller beim Einladen und Anzeigen von FMUs mit sehr vielen Parametern.

Release 0.8 mit graphischer Netzwerkdarstellung veröffentlicht

Die größte Neuerung in diesem Release ist die Unterstützung von graphischen Netzwerken und der Verknüpfung von Variablen im Diagramm. Details sind im ChangeLog.

Program versions

Installed version: ${VERSION}
Newest version: 0.9.3 (download)

News

Release 0.9.3 veröffentlicht (13.01.2022)

There were quite a few smaller improvements, mostly related to FMU debugging. Also, MasterSim is now available in a Debian package repository for Ubuntu and derived distros. The Linux desktop integration is now complete - both for the stand-alone 7z-release as for the Debian package.

Release 0.9.0 available (15.09.2021)

This release brings only smaller improvements and compatibility fixes. Details about the changes are in the change log on the download-page: https://sourceforge.net/projects/mastersim/files/releases.

Patch-Release 0.8.3 available (24.11.2020)

This is a small release, with quite a few usability improvements. And also, a German version of the manual is available and the english manual has been revised. It can be opened from within the program's help menu, or downloaded from the MasterSim documentation page.

Patch-Release 0.8.2 available (14.11.2019)

It is done - 60 pages in the first user manual (as PDF) or as online variant are available. In it are all the details and 'secrets' of the MasterSim-user interface and the simulator. The manual is available on the MasterSim documentation page. Further, all previously written publications about MasterSim are now referenced there.

Quite a few smaller bugs have been fixed (they came up while writing the manual and checking things here and there). Also, in the new release there is a neat little feature, that - I hope - you will like as well.

"Small" Patch-Release 0.8.1 available:

Just 2 days after the 0.8 release the new version has a few, but possible important feature extensions. For one thing, connections can now be assigned transformation operations (unit conversions, sign inversions) and the user interface is much more responsive when importing and handling FMUs with many exported parameters.

Release 0.8 with graphical network view available:

Greatest extension and new feature is the ability to view the simulation scenario in a network view, and draw connections in the graphical display. Also, many small usability improvements have been added. See details in the ChangeLog.
