On Linux run MasterSimulatorUI first via command line. This will indicate possibly missing 
shared libraries. Install required Qt5 packages using your package manager. Usually
this will be something like:

> sudo apt install qt5-default libqt5svg5

In the user interface you can use the option Help->Desktop Integration...
to create .desktop file (launcher/start menu link) and associate
.msim files with MasterSimulator.
